Manual Testing Checklist

Verification Steps

  • ✓ Base card styles display correctly with proper spacing
  • ✓ Glassmorphism effect shows backdrop blur (check browser support)
  • ✓ Hover effects work: transform translateY and shadow increase
  • ✓ Image zoom effect on hover works smoothly
  • ✓ Tags display correctly with proper styling
  • ✓ Responsive grid adapts: 3 cols desktop, 2 cols tablet, 1 col mobile
  • ✓ Horizontal cards stack vertically on mobile
  • ✓ Interactive cards show cursor pointer and focus outline
  • ✓ Card states (loading, disabled, selected) display correctly
  • ✓ All transitions complete within 300ms
  • ✓ Reduced motion: Test with prefers-reduced-motion enabled
  • ✓ High contrast: Test with high contrast mode enabled
  • ✓ Keyboard navigation: Tab through interactive cards
